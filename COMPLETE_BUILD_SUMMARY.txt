NEXT STEPS - GETTING STARTED
================================================================================

1. INSTALL DEPENDENCIES
   ─────────────────────
   cd D:\Finance
   npm install

   This will install all required packages:
   - React, React Router
   - Material-UI and icons
   - Firebase SDK
   - Recharts
   - date-fns

2. CREATE FIREBASE PROJECT
   ────────────────────────
   a) Go to https://console.firebase.google.com
   b) Click "Add Project"
   c) Enter project name: "personal-finance-app"
   d) Create project
   e) Create Web app (select </> icon)
   f) Copy the Firebase config

3. CONFIGURE ENVIRONMENT
   ──────────────────────
   a) Copy .env.example to .env
      cp .env.example .env

   b) Edit .env and paste Firebase credentials:
      REACT_APP_FIREBASE_API_KEY=your_api_key_here
      REACT_APP_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
      REACT_APP_FIREBASE_PROJECT_ID=your_project_id
      REACT_APP_FIREBASE_STORAGE_BUCKET=your_project.appspot.com
      REACT_APP_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
      REACT_APP_FIREBASE_APP_ID=your_app_id

4. ENABLE FIREBASE SERVICES
   ────────────────────────
   In Firebase Console:

   a) Authentication
      - Enable Email/Password
      - Enable Google
      - Add authorized domains (localhost, your-domain.com)

   b) Firestore Database
      - Create database in production mode
      - Update security rules with provided rules

   c) Hosting (Optional)
      - Enable Firebase Hosting

5. RUN LOCALLY
   ───────────
   npm start

   The app will open at http://localhost:3000

   Test by:
   - Signing up with email
   - Signing in with Google
   - Adding income/expense entries
   - Checking dashboard calculations
   - Viewing charts

6. DEPLOY TO FIREBASE
   ────────────────────
   # Build
   npm run build

   # Install Firebase CLI (one time)
   npm install -g firebase-tools
   firebase login

   # Deploy
   firebase deploy

   Your app will be live at:
   https://your-project-id.firebaseapp.com

================================================================================
TROUBLESHOOTING QUICK LINKS
================================================================================

Problem: "Firebase is not initialized"
Solution: Check .env file has correct Firebase credentials

Problem: "Cannot sign up/login"
Solution: Verify authentication is enabled in Firebase Console

Problem: "Data not saving"
Solution: Check Firestore database is created and rules are correct

Problem: "Build fails"
Solution: Run "npm install" again, clear node_modules, rebuild

Problem: "Port 3000 already in use"
Solution: Use different port: "PORT=3001 npm start"

See README.md or SETUP.md for detailed troubleshooting.
